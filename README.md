## Running

1. You need to open this project in terminal.

2. Install node modules:

with Yarn:

`yarn`

with NPM:

`npm install`

3. Build

with Yarn:

`yarn build`

with NPM:

`npm run build`

Task description:

```
Описание.
1. Вы работаете над проектом, которое представляет из себя типичное SPA
приложение. В нем также присутствует навигация через типичное меню.
2. Каждый пункт меню ведет в раздел (Section1, Section2), в котором также можно
переходить в подраздел (Section1.n, Section2.n, ..).
3. Количество подразделов динамическое, те конфигурация может приходить с
сервера (на к самой задаче это не имеет непосредственное отношение).
4. Таким образом есть корневые разделы, на которые ссылается меню, и подразделы
в каждом корневом, в которые можно переходить из корневого.
5. Каждый раздел содержит типичный авто-подгружаемый список данных (List) при
переходе в него.
6. При нажатии на элемент списка происходит навигация в подраздел, в котором также
происходит авто-загрузка списка.
7. Типичный список представляет из себя линейный список данных-массив, в котором
хранятся сущности, например {id: 1, name: ‘Peter’}.
8. Каждый список в подразделе выгружается по идентификатору при нажатии на
сущность в родительском разделе.
9. Из подраздела можно переходить обратно в родительский раздел кнопкой браузера
back. При переходе обратно, родительский список НЕ должен перегружаться, но если
перейти в родителя, а затем снова нажать в тот же Entity, список в подразделе должен
перегрузиться.
10. Меню всегда доступно.
11. При нажатии на любой корневой раздел в меню в любой момент времени список в
данном корневом разделе должен перегружаться, даже если мы уже туда заходили и
выгружали данный список в прошлый раз.
Условия.
1. Для реализации задачи нужно использовать на выбор React, Vue2.
2. Задача должна быть выполнена в TypeScript.
3. Не запрещается использовать какие либо хелперы и т.п, но ядро должно быть
написано вами.
4. Эмуляцию загрузки данных можно использовать через Promise.resolve

Пример работы программы.
1. Нажали на MenuItem1.
2. Произошел переход в Section1.1. Сработала автозагрузка List1.1.
3. Нажали на Item1, произошел переход в Section1.2. Сработала автозагрузка List1.2.
4. Нажали Back в браузере, вернулись в Section1.1. Видим список List1.1.
Авто-загрузки нет. List1.2 удалился из кучи.
5. Снова нажали на Item1, снова перешли в Section1.2. Сработала автозагрузка List1.2.
6. Нажали на Item2, произошел переход в Section1.3. Сработала автозагрузка List1.3.
7. Нажали на MenuItem2.
8. Произошел переход в Section2.1. Сработала автозагрузка List2.1.
9. List1.1, List1.2, List1.3 удалился из кучи.
10. Нажали на MenuItem1. Произошел переход в Section1.1. Сработала автозагрузка
List1.1. List2.1 удалился из кучи.
```
